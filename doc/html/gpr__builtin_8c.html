Format a string according to a list and place it in a buffer.

Note
This version of vsnprintf returns the length of the string actually written in the buffer, according to truncation
Parameters
bufThe buffer to place the result into
sizeThe size of the buffer, including the trailing '\0'
fmtThe format string to use
argsArguments for the format string
Returns
The return value is the number of characters which have been written into the buffer not including the trailing '\0'
If the size of the buffer is equal to 0, the function returns 0
If an error occured, the function returns a negative number
